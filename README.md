## ✨Clase 2:
### Temas:
 
### 1. Condicionales
* **Toma de decisiones:** Uso de `if`, `else if` y `else`.
* **Switch:** Selección de casos múltiples (ej. meses del año) y el uso de `break`.
* **Operador Ternario:** Estructura simplificada `(condición) ? true : false`.
* **Comparación:** Diferencia entre igualdad débil (`==`) y estricta (`===`).

### 2. Estructuras repetitivas
* **While:** Ejecución basada en una condición previa (pre-test).
* **Do-While:** Garantiza al menos una ejecución (post-test).
* **For:** Control de repeticiones mediante contadores.
* **For...of vs For...in:** Diferencia entre recorrer valores (arrays) y llaves (objetos).

### 3. Funciones
* **Declarativas:** Definición clásica con Hoisting.
* **Expresión y Flecha:** Sintaxis moderna `const miFuncion = () => { ... }`.
* **Ámbito (Scope):** Uso de `this` en métodos y funciones autoejecutables (IIFE).
* **Parámetros y Retorno:** Paso de datos y retorno implícito en Arrow Functions.

Ejercicios: 20

## ✨Clase 3:
### Temas:

### 1. Objetos
* **Definición:** Representación de entidades del mundo real con atributos y métodos.
* **Contexto `this`:** Uso obligatorio para acceder a propiedades desde métodos internos.
* **JSON:** Serialización con `JSON.stringify()` y parseo con `JSON.parse()`.

### 2. Spread y Desestructuración
* **Desestructuración:** Extracción de valores en variables independientes con uso de alias (`:`).
* **Operador Spread (`...`):** Clonación real de objetos y unión (merge) de múltiples estructuras.
* **Operador Rest:** Agrupación de propiedades restantes en un nuevo objeto.

### 3. Colección de Datos
* **Iteración:** Diferencia técnica entre `for...in` (llaves de objeto) y `for...of` (valores de arreglos).
* **Métodos de Object:** Uso de `Object.keys()`, `Object.values()` y `Object.entries()`.
* **Mutabilidad:** Gestión de referencias en memoria y prevención de efectos secundarios.

Ejercicios: 20
