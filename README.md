## ✨Clase 2:
### Temas:
 
### 1. Condicionales
* **Toma de decisiones:** Uso de `if`, `else if` y `else`.
* **Switch:** Selección de casos múltiples (ej. meses del año) y el uso de `break`.
* **Operador Ternario:** Estructura simplificada `(condición) ? true : false`.
* **Comparación:** Diferencia entre igualdad débil (`==`) y estricta (`===`).

### 2. Estructuras repetitivas
* **While:** Ejecución basada en una condición previa (pre-test).
* **Do-While:** Garantiza al menos una ejecución (post-test).
* **For:** Control de repeticiones mediante contadores.
* **For...of vs For...in:** Diferencia entre recorrer valores (arrays) y llaves (objetos).

### 3. Funciones
* **Declarativas:** Definición clásica con Hoisting.
* **Expresión y Flecha:** Sintaxis moderna `const miFuncion = () => { ... }`.
* **Ámbito (Scope):** Uso de `this` en métodos y funciones autoejecutables (IIFE).
* **Parámetros y Retorno:** Paso de datos y retorno implícito en Arrow Functions.

Ejercicios: 20
